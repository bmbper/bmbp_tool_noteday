# 工作记录工具 note-day-egui

#### 介绍

基于Rust、Egui构建的工作记录工具，可生成周报

#### 软件架构

开发语言： Rust

UI框架： EGUI

内存数据库： redb

#### 安装教程

1.  打包编译
```shell
cargo build --release    

```
2. 产出物 target/release/egui-noteday
